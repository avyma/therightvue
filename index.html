Chronic Conditions

Illustrate with D3, the prevalence and cost per capita for major chronic health conditions across Kentucky (2010-2017). UX/UI features drop-down menu, time slider, a dashboard complete with panels and dynamic bar graph.

Heart Conditions

Using JS, combined both prevalence, and cost per capita datasets to illustrate chronic heart conditions across Kentucky. Interactive drop-down menu with dynamic responsive legend, and ranking by counties included.

Cancer

U.S Cancer Fatalities map was fully designed in Carto. SQL/HTML were keys to efficient data table joins while creating this informative web map.

Amenities Locator

This interactive map illustrates the power of Leaflets JS. Multiple GeoJSON external data sources were accessed using JS. Minimalist code design ensures flexibility and ease-of-collaboration.

Energy Systems

Understanding the GeoJSON file structures while using Leaflets JS to visualize data into layers resulted in this attractive map.

Crater

Knowledge in MapBox, PostGIS, GDAL and QGIS were essential in the creation of this map. USGS 3DEP online tools were used for raster elevation image.

Appalachian Trail

QGIS GDAL Georeferencer was used in the creation of this map. Selective rivers, lakes, water and mountains were digitized with approapriate attributes. The North America Albers Equal Area Conic (EPSG:102008) was used
